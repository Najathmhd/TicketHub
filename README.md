6.2 Installation & Setup
Prerequisites:

.NET 8.0 SDK (or later)
Microsoft SQL Server
Configuration:

Update the database connection string in appsettings.json if necessary.
Database Initialization:

Open terminal in the project folder.
Run: dotnet ef database update to create the database schema.
Run Application:

Execute: dotnet run or press F5 in Visual Studio.
Navigate to: https://localhost:7154 (or the port shown in terminal).
